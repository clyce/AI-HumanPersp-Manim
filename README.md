# 人的视角 —— (重新) 理解现代人工智能 Manim 工程

## 安装

1. 安装 [Manim](https://docs.manim.community/en/stable/installation/uv.html) （须包含 LaTeX 支持）
2. 安装 [Manim Slides](https://www.manim.community/plugin/manim-slides/)

## 运行

渲染 Slide：
```bash
manim-slides render {filename}.py {ClassName}
```

展示 Slide:
```bash
manim-slides {ClassName}
```

## 教学路径大纲

### 核心主线

用**人的感知和思维方式**重新理解 AI 的核心概念，从哲学基础出发，经由记忆模型、表征学习，最终抵达现代生成模型。强调「为什么」而非「怎么做」。

### 已完成章节

#### 00-Prepare：视频介绍
- 核心冲突：「控制 vs 涌现」
- 目标受众与预期
- 系列视频的整体规划

#### 01-Prelude：哲学基础
- 电磁波与感知的边界
- 编码与信息实在论
- 智能作为热传递的类比

#### 02-RAMvsCAM：存储 vs 记忆
- SC01: RAM 概念——日常类比 + 数组寻址
- SC02: 转折——记忆并非基于逻辑与控制
- SC03: CAM 概念——联想记忆、内容寻址
- SC04: 「拉帮结派」类比——全连接网络与情绪同步
- SC05: 初始 Hopfield 网络——两个模式的训练与回忆
- SC06: 自组织学习——Hebbian 规则的本质 + Hopfield 命名
- SC07: 记忆混乱——Hopfield 容量极限 + "它能创造新东西吗？"
- SC08: 从记忆到理解——BM 作为生成模型、HF(CAM) vs BM(Generative) 范式转换

### 后续章节规划

#### 03：概念如何从现象诞生？
- **主题**: 表征学习与特征提取
- **核心问题**: 大脑（和机器）如何从原始感知数据中提炼出「概念」？
- **内容规划**:
  - 从像素到特征：边缘、纹理、形状的层级抽象
  - 自编码器（Autoencoder）：学习压缩表示
  - RBM 简介：BM 的结构约束如何简化训练（承接 02-SC08 的一句话铺垫）
  - Deep Belief Network：逐层贪婪训练的直觉
- **教学风格**: 延续「辩论 → 记忆」类比，引入「摘要 → 理解」类比

#### 04：常见连接主义模型及其设计视角
- **主题**: CNN / RNN / Transformer 的「为什么」
- **核心问题**: 为什么要设计这样的结构？它们解决了什么根本问题？
- **内容规划**:
  - CNN：平移不变性 → 局部感受野 → 为什么卷积是合理的
  - RNN：时间序列 → 记忆的延续 → 为什么循环是自然的
  - Attention / Transformer：全局关联 → 为什么 self-attention 有效
  - 反思：每种结构都是对「世界如何运作」的一种假设
- **教学风格**: 从认知科学的视角引入每种结构的设计动机

#### 05：从现象编码到抽象
- **主题**: 深度学习如何逐层抽象
- **核心问题**: 机器是如何从「看到什么」过渡到「理解什么」的？
- **内容规划**:
  - 特征可视化：每一层在「看」什么
  - 表征空间的几何结构
  - 迁移学习：为什么低层特征可以共享
  - 涌现能力：规模如何带来质变
- **教学风格**: 大量可视化，将抽象概念锚定到直觉

#### 06：生成与创造
- **主题**: 从 BM/VAE 到扩散模型
- **核心问题**: 机器如何「创造」从未见过的东西？
- **内容规划**:
  - 生成 vs 判别：两种认知范式
  - VAE：隐空间的结构 → 插值与生成
  - GAN：对抗博弈 → Nash 均衡的直觉
  - 扩散模型：从噪声到结构 → 与热力学的深层类比
  - 回归主线：涌现、自组织、与 01-Prelude 的呼应
- **教学风格**: 回到哲学基础，闭合整个叙事环

### 贯穿全系列的设计原则

1. **先「为什么」后「怎么做」**: 每个模型先讲设计动机，再讲实现细节
2. **类比驱动**: 用日常经验和认知科学锚定技术概念
3. **诚实展示**: 实验结果不作假，模型局限不回避
4. **叙事连贯**: 每章结尾铺垫下一章，形成完整的认知链条
5. **视觉优先**: Manim 动画承载核心论证，文字辅助
